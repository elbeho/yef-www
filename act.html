[% META title = 'Information for Organizers - Using Act' %]

Act is a conference management toolkit originaly developped by the French Perl Mongers after the YAPC::Europe 2003 conference.

The YAPC::Europe Foundation online payment system can be connected to a conference managed by Act. Though it should be possible to install Act on any system (the source code is available), the YAPC::Eureop online payment system is only configured on the Act instance hosted by the French Perl Mongers.

If you want your conference web site to be hosted on the official Act platform (and possibly use the YAPC::Europe online payment system), you have to request the creation of a conference web site directly to the Act team. The best way to do this is to subscribe to the Act users mailing-list at http://listes.mongueurs.net/mailman/listinfo/act.
