The whole board (officials + coops) decides on issues like these, and others:

  • Install a committee, dissolve a committee.

  • Appoint or dismiss committee members.

  • Vote on proposals made by committees.

  • Ask a committee to investigate an issue and make a proposal.

  • Set a deadline for any discussion to be held by the board or by a committee.
